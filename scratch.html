Scratch

Este foi meu projeto desenvolvido em scratch.

Gato Rato v1

No jogo, inspirado pela brincadeira de gato e rato de criança onde uma fica no meio de um circulo e os outros a impedem de escapar, o rato tem que tentar pegar a maçã e fugir do círculo sem encostar no gato.

Gato Rato v2

Para esta versão o jogo principal foi mantido o mesmo e uma tela inicial perguntado o nome do usuário foi adicionado para Exibir uma mensagem personalizada de inicio de jogo.

Gato Rato entrega 2

Para esta entrega, uma funcionalidade de escolher entre várias linguas (PT/EN/ES/DE) que determina vozes para as entidades foi adicionado para internacionalizar o jogo.
